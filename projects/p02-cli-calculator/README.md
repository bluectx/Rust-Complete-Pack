# Projet 2: CLI Calculator

## Description

Une calculatrice en ligne de commande qui effectue des opérations arithmétiques de base.

## Objectifs d'Apprentissage

- Parser des arguments CLI
- Gérer les erreurs avec Result et anyhow
- Implémenter des opérations mathématiques
- Gérer la division par zéro
- Écrire des tests complets

## Utilisation

```bash
cargo run -- 5 + 3
cargo run -- 10 - 4
cargo run -- 3 * 4
cargo run -- 15 / 3
```

## Tests

```bash
cargo test
```
