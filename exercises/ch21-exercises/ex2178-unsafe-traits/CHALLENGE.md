# Unsafe traits

**Level:** ⭐⭐⭐ ADVANCED

**Objective:** Implémenter un unsafe trait

**Problem:**

Créez `unsafe trait UnsafeTrait { }` et implémentez-le avec unsafe impl.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Shift de responsabilité au programmeur.
- unsafe trait = contrat que l'implémenteur doit honorer.
