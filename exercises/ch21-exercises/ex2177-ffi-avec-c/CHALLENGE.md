# FFI avec C

**Level:** ⭐⭐⭐ ADVANCED

**Objective:** Appeler du code C depuis Rust

**Problem:**

Utilisez `extern "C" { fn c_function(); }` pour linker avec une lib C.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Interopérabilité avec C et les libs natives.
- extern "C" declare une fonction C à appeler.
