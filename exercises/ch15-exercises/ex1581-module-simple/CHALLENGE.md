# Module simple

**Level:** ⭐ BEGINNER

**Objective:** Organiser le code en modules

**Problem:**

Créez un module math { } et définissez une fonction à l'intérieur. Appelez-la.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Les modules créent des namespaces.
- `mod name { }`
