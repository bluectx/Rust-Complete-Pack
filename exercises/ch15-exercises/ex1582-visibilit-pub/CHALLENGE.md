# Visibilité pub

**Level:** ⭐ BEGINNER

**Objective:** Rendre les items visibles en dehors du module

**Problem:**

Déclarez une fonction pub dans un module et appelez-la en dehors.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Par défaut, tout est privé (encapsulation).
- `pub fn ...` rend l'item public.
