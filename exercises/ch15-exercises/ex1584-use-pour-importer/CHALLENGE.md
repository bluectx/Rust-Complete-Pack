# use pour importer

**Level:** ⭐ BEGINNER

**Objective:** Importer des items avec use

**Problem:**

Utilisez `use module::function;` puis appelez function() directement.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- use réduit la verbosité.
- `use path::item;` importe l'item dans le scope actuel.
