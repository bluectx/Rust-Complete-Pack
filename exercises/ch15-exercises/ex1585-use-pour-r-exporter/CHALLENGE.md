# use :: pour réexporter

**Level:** ⭐ BEGINNER

**Objective:** Réexporter des items pub use

**Problem:**

Créez une public re-export avec `pub use`.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Utile pour créer des APIs conviviales.
- `pub use path::item;` exporte item comme si c'était défini localement.
