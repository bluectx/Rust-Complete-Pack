# Function-like macros

**Level:** ⭐⭐⭐ ADVANCED

**Objective:** Écrire une proc-macro qui fonctionne comme une fonction

**Problem:**

Créez une macro qui prend du code et le transforme.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Maximum de flexibilité syntaxique.
- Ressemble à des function calls syntaxiquement.
