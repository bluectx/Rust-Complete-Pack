# syn et quote

**Level:** ⭐⭐⭐ ADVANCED

**Objective:** Utiliser les crates syn et quote pour parser et générer du code

**Problem:**

Parsez un input TokenStream avec syn et généreza avec quote!.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Outils standards pour les proc-macros.
- syn = parsing ; quote = génération de code.
