# println! et vec! comme exemples

**Level:** ⭐⭐ INTERMEDIATE

**Objective:** Comprendre comment les macros de stdlib fonctionnent

**Problem:**

Tracez comment `vec![1, 2, 3]` s'expande.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Les macros stdlib sont des exemples à apprendre.
- vec! est une macro built-in ; on peut lire sa définition.
