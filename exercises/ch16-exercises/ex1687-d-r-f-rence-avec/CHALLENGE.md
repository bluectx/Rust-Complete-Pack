# Déréférence avec *

**Level:** ⭐ BEGINNER

**Objective:** Accéder à la valeur d'une Box

**Problem:**

Dérérencez une Box avec * pour accéder à la valeur.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Déréférence = accès indirect.
- `*box` accède à la valeur pointée.
