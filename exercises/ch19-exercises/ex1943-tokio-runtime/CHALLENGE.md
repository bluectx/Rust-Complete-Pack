# tokio runtime

**Level:** ⭐⭐ INTERMEDIATE

**Objective:** Utiliser tokio comme runtime async

**Problem:**

Utilisez `#[tokio::main]` et lancez une tâche async.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- tokio gère les threads et les tâches.
- #[tokio::main] initialise le runtime.
