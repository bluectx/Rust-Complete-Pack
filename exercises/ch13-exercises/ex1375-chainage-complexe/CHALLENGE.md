# Chainage complexe

**Level:** ⭐ BEGINNER

**Objective:** Combiner map, filter, et d'autres adaptateurs

**Problem:**

Créez un vec, doublez, filtrez, collectez. Affichez le résultat.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- C'est l'approche fonctionnelle idiomatique de Rust.
- Les adaptateurs d'itérateurs se chainent élégamment.
