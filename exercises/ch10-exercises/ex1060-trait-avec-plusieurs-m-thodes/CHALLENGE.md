# Trait avec plusieurs méthodes

**Level:** ⭐ BEGINNER

**Objective:** Gérer des traits avec plusieurs signatures

**Problem:**

Créez un trait Formes avec aire(&self) -> f64 et perimetres(&self) -> f64.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Tous les implémenteurs doivent fournir tous les méthodes.
- Un trait peut avoir plusieurs méthodes.
