# Trait basique

**Level:** ⭐ BEGINNER

**Objective:** Définir un trait et implémenter ses méthodes

**Problem:**

Créez un trait Animal avec méthode faire_bruit(&self). Implémentez-le pour Chien et Chat.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Les traits définissent un contrat que les types doivent respecter.
- `trait Name { fn method(&self); }`
