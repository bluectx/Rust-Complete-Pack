# Trait avec méthode par défaut

**Level:** ⭐ BEGINNER

**Objective:** Fournir une implémentation par défaut dans un trait

**Problem:**

Ajoutez une méthode avec implémentation par défaut dans Animal.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Les implémenteurs peuvent utiliser le défaut ou l'override.
- `fn method(&self) { /* code */ }` dans le trait.
