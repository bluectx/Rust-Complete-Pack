# Property-based testing avec quickcheck

**Level:** ⭐⭐⭐ ADVANCED

**Objective:** Tester avec des propriétés plutôt que des cas spécifiques

**Problem:**

Utilisez quickcheck pour générer des test cases aléatoires.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Couvre plus de cas qu'on ne pourrait écrire manuellement.
- Property-based testing = générer beaucoup de cas de test.
