# Fonction de test basique

**Level:** ⭐ BEGINNER

**Objective:** Écrire et exécuter un test unitaire

**Problem:**

Écrivez `#[test] fn it_works() { assert!(true); }` et lancez `cargo test`.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- `cargo test` exécute tous les tests.
- `#[test]` marque une fonction comme test.
