# if comme expression

**Level:** ⭐ BEGINNER

**Objective:** Utiliser if pour assigner une valeur à une variable

**Problem:**

Utilisez if-else pour déterminer le maximum de deux nombres et assignez le résultat.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- `let x = if condition { val1 } else { val2 };`
- if peut retourner une valeur ; les deux branches doivent retourner le même type.
