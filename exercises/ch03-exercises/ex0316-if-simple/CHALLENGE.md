# if simple

**Level:** ⭐ BEGINNER

**Objective:** Exécuter du code de manière conditionnelle avec if

**Problem:**

Écrivez une branche if qui affiche un message si une variable est > 10.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- La condition doit être un bool, pas un entier.
- if condition { // code } exécute le code si la condition est true.
