# Associated constants

**Level:** ⭐⭐ INTERMEDIATE

**Objective:** Ajouter des constantes dans les traits

**Problem:**

Écrivez `trait Foo { const MAGIC: i32; }`.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Généralisation des associated types.
- const dans les traits doivent être fournis par l'implémenteur.
