# Closure simple

**Level:** ⭐ BEGINNER

**Objective:** Définir et utiliser une fonction anonyme

**Problem:**

Créez une closure qui ajoute 1 à son argument. Appelez-la.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Les closures capturent les variables de l'environnement.
- `let f = |x| x + 1;`
