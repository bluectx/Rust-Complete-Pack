# Boucle infinie loop

**Level:** ⭐ BEGINNER

**Objective:** Utiliser `loop` avec `break`

**Problem:**

Écrivez une boucle infinie qui compte de 1 à 5, puis s'arrête.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- break termine la boucle ; continue passe à la prochaine itération.
- loop { } répète indéfiniment jusqu'à un break.
