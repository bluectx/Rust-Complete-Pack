# Boucles imbriquées

**Level:** ⭐ BEGINNER

**Objective:** Imbriquer des boucles for

**Problem:**

Affichez une table de multiplication 5x5 avec des boucles imbriquées.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- break et continue s'appliquent à la boucle la plus interne.
- Les boucles peuvent contenir d'autres boucles.
