# Accès aux champs

**Level:** ⭐ BEGINNER

**Objective:** Accéder et modifier les champs d'une struct

**Problem:**

Créez une instance, modifiez un champ, affichez un autre champ.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Pour modifier, la variable instance doit être mutable.
- Accès via dot notation : `instance.field`
