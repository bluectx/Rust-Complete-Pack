# Associated types

**Level:** ⭐⭐ INTERMEDIATE

**Objective:** Utiliser des types associés dans les traits

**Problem:**

Créez un trait avec `type Item;` et implémentez-le.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Plus flexible que les génériques pour certains cas.
- `type Item;` définit un type qui l'implémenteur doit spécifier.
