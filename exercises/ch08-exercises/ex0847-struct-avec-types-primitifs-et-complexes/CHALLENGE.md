# Struct avec types primitifs et complexes

**Level:** ⭐ BEGINNER

**Objective:** Mélanger différents types dans une struct

**Problem:**

Créez une struct Livre avec titre: String, pages: u32, disponible: bool.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- La composition est plus flexible que l'héritage.
- Les structs peuvent contenir n'importe quel type.
