# Impl basique (méthodes)

**Level:** ⭐ BEGINNER

**Objective:** Ajouter des méthodes à une struct

**Problem:**

Définissez Personne, puis un impl Personne { } avec une méthode saluer(&self).

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- &self = méthode qui n'emprute qu'en lecture.
- `impl StructName { fn method(&self) { } }`
