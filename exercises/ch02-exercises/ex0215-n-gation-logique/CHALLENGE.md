# Négation logique

**Level:** ⭐ BEGINNER

**Objective:** Utiliser ! pour inverser une condition

**Problem:**

Appliquez ! à plusieurs expressions booléennes.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- La négation s'applique avant && et ||.
- ! inverse true en false et vice-versa.
