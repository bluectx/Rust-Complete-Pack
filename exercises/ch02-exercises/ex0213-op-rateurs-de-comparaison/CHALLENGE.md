# Opérateurs de comparaison

**Level:** ⭐ BEGINNER

**Objective:** Utiliser ==, !=, <, >, <=, >=

**Problem:**

Comparez plusieurs paires de nombres et affichez les résultats booléens.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Tous les types comparables supportent ces opérateurs.
- Les opérateurs de comparaison retournent bool.
