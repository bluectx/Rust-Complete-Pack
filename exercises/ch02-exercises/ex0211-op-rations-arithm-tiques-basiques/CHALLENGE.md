# Opérations arithmétiques basiques

**Level:** ⭐ BEGINNER

**Objective:** Effectuer +, -, *, /, % sur des entiers

**Problem:**

Calculez 15 + 3, 20 - 8, 4 * 7, 20 / 3, 20 % 3. Commentez sur la division.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- La division entière tronque ; utilisez f64 pour garder les décimales.
- / divise les entiers sans reste.
