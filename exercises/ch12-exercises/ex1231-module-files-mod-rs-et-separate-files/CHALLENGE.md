# Module files (mod.rs et separate files)

**Level:** ⭐⭐ INTERMEDIATE

**Objective:** Organiser les modules sur le disque

**Problem:**

Créez `src/math/mod.rs` et référencez-le avec `mod math;`.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Scalabilité de la structure du projet.
- Les modules peuvent vivre dans des fichiers separés.
