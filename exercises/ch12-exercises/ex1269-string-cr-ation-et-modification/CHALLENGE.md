# String - création et modification

**Level:** ⭐ BEGINNER

**Objective:** Travailler avec les Strings

**Problem:**

Créez une String, utilisez push_str() pour ajouter, affichez.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- String est mutable et peut grandir dynamiquement.
- `let mut s = String::new();` ou `let s = String::from("texte");`
