# Vec - créer et ajouter

**Level:** ⭐ BEGINNER

**Objective:** Utiliser un vecteur et ajouter des éléments

**Problem:**

Créez un Vec<i32>, poussez 3 nombres, affichez la longueur.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Vec est mutable ; push() ajoute des éléments.
- `let mut v = Vec::new();` ou `let v = vec![1, 2, 3];`
