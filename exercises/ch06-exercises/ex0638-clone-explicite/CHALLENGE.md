# Clone explicite

**Level:** ⭐ BEGINNER

**Objective:** Cloner une valeur pour éviter le déplacement

**Problem:**

Utilisez `.clone()` sur une String pour la copier complètement.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- clone() est explicite mais potentiellement coûteux en performances.
- clone() crée une profonde copie d'une valeur.
