# Builder pattern

**Level:** ⭐⭐ INTERMEDIATE

**Objective:** Implémenter le builder pattern pour une configuration complexe

**Problem:**

Créez une struct Client avec un ClientBuilder qui construit pas à pas.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Idiom Rust pour les constructeurs complexes.
- Builder = méthodes chainées pour construire.
