# Strategy pattern

**Level:** ⭐⭐ INTERMEDIATE

**Objective:** Utiliser le strategy pattern avec traits

**Problem:**

Définissez un trait Strategy et plusieurs implémentations.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Polymorphisme via traits.
- Strategy = différentes implémentations d'une interface.
