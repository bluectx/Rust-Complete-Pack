# Custom Display impl

**Level:** ⭐⭐ INTERMEDIATE

**Objective:** Implémenter Display pour un type custom

**Problem:**

Implémentez `impl fmt::Display for MyType { }`.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Personnalisez la sérialisation texte.
- Display est utilisé par println! et to_string().
