# Feature gates

**Level:** ⭐⭐ INTERMEDIATE

**Objective:** Utiliser les features conditionnelles

**Problem:**

Écrivez `#[cfg(feature = "feature_name")]` et compilez avec flags.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Modularité et options de compilation.
- Les features permettent de compilation conditionnelle.
