# Benchmarking basique

**Level:** ⭐⭐ INTERMEDIATE

**Objective:** Mesurer la performance d'une fonction

**Problem:**

Utilisez `#![feature(test)]` et `#[bench]` pour benchmarker.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Ne pas optimiser sans mesures.
- Benching mesure la performance relative.
