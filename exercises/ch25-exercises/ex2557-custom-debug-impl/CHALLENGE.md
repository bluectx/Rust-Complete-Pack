# Custom Debug impl

**Level:** ⭐⭐ INTERMEDIATE

**Objective:** Implémenter Debug pour un type custom

**Problem:**

Implémentez `impl fmt::Debug for MyType { }`.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Utile pour le débogage.
- Debug est utilisé par {:?} et {:?} dans println!.
