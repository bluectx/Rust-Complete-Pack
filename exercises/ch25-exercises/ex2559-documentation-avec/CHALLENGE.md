# Documentation avec ///

**Level:** ⭐⭐ INTERMEDIATE

**Objective:** Écrire de la documentation pour une fonction

**Problem:**

Documentez une fonction avec `/// Docs` et lancez `cargo doc`.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Documentation compilée en HTML.
- /// est la doc interne ; //! est la doc de crate.
