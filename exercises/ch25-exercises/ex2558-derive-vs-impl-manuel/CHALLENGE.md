# derive vs impl manuel

**Level:** ⭐⭐ INTERMEDIATE

**Objective:** Savoir quand utiliser #[derive] vs impl manuel

**Problem:**

Comparez `#[derive(Debug)]` vs `impl Debug`.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Pragmatisme : utilisez derive quand c'est possible.
- #[derive] génère automatiquement ; impl offre plus de contrôle.
