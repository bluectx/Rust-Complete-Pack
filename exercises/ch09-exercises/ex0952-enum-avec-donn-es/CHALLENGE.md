# Enum avec données

**Level:** ⭐ BEGINNER

**Objective:** Associer des données à des variantes

**Problem:**

Créez un enum Message avec variantes Text(String), Nombre(i32), Booleen(bool).

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Chaque variante peut avoir ses propres données.
- `enum Name { Variant1(Type1), Variant2(Type2), ... }`
