# Type-driven development

**Level:** ⭐⭐⭐ ADVANCED

**Objective:** Concevoir avec les types pour éviter les bugs

**Problem:**

Créez une struct avec des types spécifiques pour empêcher les états invalides.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- "Make invalid states unrepresentable".
- Types encodent des invariants.
