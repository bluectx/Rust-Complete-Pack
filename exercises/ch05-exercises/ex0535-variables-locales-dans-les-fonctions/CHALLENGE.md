# Variables locales dans les fonctions

**Level:** ⭐ BEGINNER

**Objective:** Comprendre la portée des variables dans les fonctions

**Problem:**

Déclarez une variable dans une fonction et essayez d'y accéder en dehors.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- La portée commence à sa déclaration et se termine à la fin du bloc.
- Les variables déclarées dans une fonction ont une portée locale.
