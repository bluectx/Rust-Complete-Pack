# Annotation de type de retour

**Level:** ⭐ BEGINNER

**Objective:** Spécifier le type de retour d'une fonction

**Problem:**

Écrivez une fonction multiply(a: i32, b: i32) -> i32 { }

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- La flèche -> indique le type de retour.
- `fn nom(params) -> Type { }`
