# Fonction avec return

**Level:** ⭐ BEGINNER

**Objective:** Retourner une valeur d'une fonction

**Problem:**

Écrivez une fonction qui calcule le carré et retourne le résultat.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Une fonction retourne soit rien (unit type), soit une valeur spécifiée.
- Soit avec `return x;`, soit en mettant x comme dernière ligne sans ;
