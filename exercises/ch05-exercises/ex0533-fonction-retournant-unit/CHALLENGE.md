# Fonction retournant unit ()

**Level:** ⭐ BEGINNER

**Objective:** Comprendre que les fonctions sans return retournent ()

**Problem:**

Écrivez une fonction qui affiche un message et ne retourne rien.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- `return;` avec ; est une instruction, pas une expression.
- Les fonctions sans -> Type retournent implicitement () (unit).
