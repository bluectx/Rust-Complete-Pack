# Vec - itération

**Level:** ⭐ BEGINNER

**Objective:** Boucler sur un vecteur

**Problem:**

Créez un vec et affichez tous les éléments avec une boucle for.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- & emprunte le vec immuablement ; move le consomme.
- `for elem in &v { }`
