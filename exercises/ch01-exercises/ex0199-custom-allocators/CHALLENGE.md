# Custom allocators

**Level:** ⭐ BEGINNER

**Objective:** Écrire un allocateur mémoire custom

**Problem:**

Implémentez un GlobalAlloc custom (éducationnel).

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Contrôle total sur la mémoire.
- GlobalAlloc contrôle l'allocation mémoire.
