# Tests d'intégration

**Level:** ⭐ BEGINNER

**Objective:** Écrire des tests d'intégration dans tests/ directory

**Problem:**

Créez `tests/integration_test.rs` et testez la API publique.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Différence entre unit et integration tests.
- tests/ contient les tests d'intégration (black-box).
