# Glob imports

**Level:** ⭐ BEGINNER

**Objective:** Importer tous les items d'un module

**Problem:**

Utilisez `use module::*;` pour importer tous les public items.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Pratique mais peut causer des conflits de noms.
- `*` est un glob mais doit être utilisé avec soin.
