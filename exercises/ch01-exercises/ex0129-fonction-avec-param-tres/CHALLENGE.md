# Fonction avec paramètres

**Level:** ⭐ BEGINNER

**Objective:** Passer des arguments à une fonction

**Problem:**

Écrivez une fonction qui prend un entier et affiche son carré.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Les paramètres doivent avoir une annotation de type.
- `fn nom(param: Type) { }`
