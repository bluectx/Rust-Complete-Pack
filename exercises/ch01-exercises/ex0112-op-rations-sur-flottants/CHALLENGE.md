# Opérations sur flottants

**Level:** ⭐ BEGINNER

**Objective:** Effectuer des opérations arithmétiques sur des f64

**Problem:**

Calculez 10.5 / 3.0, comparez avec la division entière équivalente.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Les résultats flottants peuvent avoir des erreurs d'arrondi.
- Les opérations sur flottants conservent les décimales.
