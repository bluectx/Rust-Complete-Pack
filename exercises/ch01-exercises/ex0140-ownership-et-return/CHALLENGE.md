# Ownership et return

**Level:** ⭐ BEGINNER

**Objective:** Retourner une valeur pour transférer l'ownership

**Problem:**

Écrivez une fonction qui retourne une String. Assignez le résultat à une nouvelle variable.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- On peut recréer l'ownership en retournant.
- Retourner une value transfère l'ownership au appelant.
