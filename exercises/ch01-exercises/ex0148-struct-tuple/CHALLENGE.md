# Struct tuple

**Level:** ⭐ BEGINNER

**Objective:** Utiliser des structs tuple (format simplifié)

**Problem:**

Créez une struct tuple Point(f64, f64) et accédez à ses champs par index.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Les structs tuple sont plus légères pour de simples groupes.
- `struct Name(Type1, Type2, ...)`
