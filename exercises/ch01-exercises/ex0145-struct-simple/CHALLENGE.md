# Struct simple

**Level:** ⭐ BEGINNER

**Objective:** Définir et instancier une struct

**Problem:**

Définissez une struct Personne avec nom: String et age: u32. Instanciez-la.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Les structs groupent des données liées.
- `struct Name { field: Type, ... }`
