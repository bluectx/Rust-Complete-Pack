# Lock-free programming

**Level:** ⭐ BEGINNER

**Objective:** Écrire du code sans locks avec atomics

**Problem:**

Utilisez std::sync::atomic pour des structures lock-free.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Haute performance sans les dangers des locks.
- Lock-free = plus performant mais subtil.
