# Référence mutable

**Level:** ⭐ BEGINNER

**Objective:** Emprunter une valeur mutablement et la modifier

**Problem:**

Créez une référence mutable avec &mut et modifiez la valeur via la référence.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- La variable originale doit être mutable ; une seule &mut peut exister à la fois.
- &mut crée une référence mutable ; on peut modifier via la référence.
