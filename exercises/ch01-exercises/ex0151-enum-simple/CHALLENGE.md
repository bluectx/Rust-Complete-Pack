# Enum simple

**Level:** ⭐ BEGINNER

**Objective:** Définir une énumération avec plusieurs variantes

**Problem:**

Créez un enum Couleur avec Rouge, Vert, Bleu. Instanciez chaque variante.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Les enums représent des types qui peuvent être dans plusieurs états.
- `enum Name { Variant1, Variant2, ... }`
