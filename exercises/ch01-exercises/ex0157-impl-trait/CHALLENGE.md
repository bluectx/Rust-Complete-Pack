# Impl trait

**Level:** ⭐ BEGINNER

**Objective:** Implémenter un trait pour une struct

**Problem:**

Implémentez Animal pour Chien avec sa propre logique de faire_bruit.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Chaque type peut implémenter un trait différemment.
- `impl TraitName for StructName { fn method(&self) { } }`
