# Mocking et testing avec dyn traits

**Level:** ⭐ BEGINNER

**Objective:** Mocke des dépendances pour tester

**Problem:**

Créez des mock implementations d'un trait pour les tests.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Testabilité via l'abstraction.
- Traits permettent le mocking facile.
