# Raw pointers (*const et *mut)

**Level:** ⭐ BEGINNER

**Objective:** Manipuler les pointeurs bruts

**Problem:**

Créez un *const T, un *mut T, et dérérencez-les dans unsafe.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Aucune garantie de sécurité ; utilisez avec prudence.
- Raw pointers contournent le borrow checker.
