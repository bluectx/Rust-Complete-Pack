# Types numériques entiers

**Level:** ⭐ BEGINNER

**Objective:** Comprendre i32, u32, i64, u8 et leurs plages

**Problem:**

Créez des variables de différents types entiers (i8, u8, i32, u64). Tentez d'assigner une valeur hors plage à u8.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Le type par défaut pour un entier littéral est i32.
- Rust a plusieurs types entiers signés (i) et non signés (u) de différentes tailles.
