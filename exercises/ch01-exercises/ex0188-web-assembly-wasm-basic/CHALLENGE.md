# Web Assembly (WASM) basic

**Level:** ⭐ BEGINNER

**Objective:** Compiler Rust en WASM

**Problem:**

Utilisez wasm-pack pour compiler une lib Rust en WASM.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Rust excelle pour le WASM.
- WASM = bytecode exécutable dans les navigateurs.
