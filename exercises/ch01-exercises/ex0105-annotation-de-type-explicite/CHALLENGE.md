# Annotation de type explicite

**Level:** ⭐ BEGINNER

**Objective:** Annoter explicitement le type d'une variable

**Problem:**

Déclarez une variable avec annotation de type explicite. Comparez avec l'inférence.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Syntaxe : `let x: Type = valeur;`
- Bien que l'inférence soit possible, les annotations peuvent clarifier l'intention.
