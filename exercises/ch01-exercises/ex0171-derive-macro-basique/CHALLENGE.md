# Derive macro basique

**Level:** ⭐ BEGINNER

**Objective:** Écrire une derive macro avec proc_macro

**Problem:**

Créez une proc-macro crate et écrivez un #[derive(MyMacro)].

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Bien plus puissantes que les macros déclaratives.
- Derive macros transforment les structures à la compilation.
