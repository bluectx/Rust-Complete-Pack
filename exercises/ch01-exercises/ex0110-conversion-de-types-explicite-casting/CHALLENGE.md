# Conversion de types explicite (casting)

**Level:** ⭐ BEGINNER

**Objective:** Utiliser `as` pour convertir entre types

**Problem:**

Convertissez une variable i32 en u16, puis en f64. Affichez les résultats.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Le casting peut causer une perte de données si mal effectué.
- Syntaxe : `variable as TypeCible`
