# Type caractère

**Level:** ⭐ BEGINNER

**Objective:** Comprendre le type `char` et Unicode

**Problem:**

Assignez des caractères à une variable, y compris des caractères Unicode (émojis, accents).

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- char utilise des guillemets simples, String utilise des guillemets doubles.
- char représente un caractère Unicode simple (4 bytes).
