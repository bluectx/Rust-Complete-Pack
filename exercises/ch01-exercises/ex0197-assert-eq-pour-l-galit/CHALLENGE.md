# assert_eq! pour l'égalité

**Level:** ⭐ BEGINNER

**Objective:** Tester l'égalité avec assert_eq!

**Problem:**

Écrivez un test qui vérifie `assert_eq!(2 + 2, 4);`

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- assert_eq! affiche les deux valeurs en cas d'échec.
- `assert_eq!(left, right)` panique si left != right.
