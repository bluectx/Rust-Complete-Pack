# Inférence de types

**Level:** ⭐ BEGINNER

**Objective:** Laisser Rust inférer le type d'une variable

**Problem:**

Créez des variables sans spécifier leur type. Rust les infère automatiquement.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- On peut souvent omettre les annotations de type.
- Rust a un puissant système d'inférence de types.
