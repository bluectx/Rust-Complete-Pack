# Fonction avec plusieurs paramètres

**Level:** ⭐ BEGINNER

**Objective:** Gérer plusieurs paramètres

**Problem:**

Écrivez une fonction add qui prend deux entiers et affiche leur somme.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Chaque paramètre doit avoir un type.
- Séparez les paramètres par des virgules.
