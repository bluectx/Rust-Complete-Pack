# Macro recursion

**Level:** ⭐ BEGINNER

**Objective:** Créer une macro qui s'appelle elle-même

**Problem:**

Écrivez une macro qui traite des listes (item1, item2, ...).

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Puissance des macros déclaratives.
- Les macros peuvent recursify pour traiter les listes.
