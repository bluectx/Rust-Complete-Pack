# tokio::spawn pour spawner des tâches

**Level:** ⭐ BEGINNER

**Objective:** Exécuter plusieurs futures concurremment

**Problem:**

Utilisez `tokio::spawn()` pour lancer plusieurs tâches.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Concurrence sans threading overhead.
- spawn crée une tâche independante.
