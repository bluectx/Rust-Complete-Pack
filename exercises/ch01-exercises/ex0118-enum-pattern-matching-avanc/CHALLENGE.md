# Enum pattern matching avancé

**Level:** ⭐ BEGINNER

**Objective:** Pattern matching complexes

**Problem:**

Matchez des enums imbriqués avec gardes (if conditions).

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Les gardes offrent plus de contrôle.
- `match { Pat if cond => ... }` ajoute des conditions.
