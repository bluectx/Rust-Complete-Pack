# Fonction simple sans paramètres

**Level:** ⭐ BEGINNER

**Objective:** Déclarer et appeler une fonction basique

**Problem:**

Écrivez une fonction qui affiche "Hello, Rust!" et appelez-la.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Les fonctions en Rust utilisent fn, pas function ou def.
- `fn nom() { // code }`
