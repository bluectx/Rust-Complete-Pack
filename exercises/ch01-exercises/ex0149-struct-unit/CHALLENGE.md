# Struct unité

**Level:** ⭐ BEGINNER

**Objective:** Comprendre les structs sans champs

**Problem:**

Définissez une struct unité Marker; et instanciez-la.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Utile pour le typage et les markers génériques.
- `struct Name;` définit une struct sans champs.
