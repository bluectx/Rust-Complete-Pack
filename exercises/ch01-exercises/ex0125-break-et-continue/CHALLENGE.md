# break et continue

**Level:** ⭐ BEGINNER

**Objective:** Contrôler le flux de boucles avec break et continue

**Problem:**

Comptez de 1 à 10, ignorez 5 avec continue, arrêtez à 8 avec break.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Utilisez ces mots-clés pour un contrôle fin.
- continue saute le reste de l'itération ; break sort de la boucle.
