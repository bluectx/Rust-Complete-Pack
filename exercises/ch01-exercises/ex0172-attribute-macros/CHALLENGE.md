# Attribute macros

**Level:** ⭐ BEGINNER

**Objective:** Écrire une attribute macro

**Problem:**

Créez un #[my_attr] qui transforme une fonction.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Générer du code à partir d'annotations.
- Attribute macros modifient ou remplacent les éléments.
