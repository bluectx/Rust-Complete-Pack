# Chemin absolu et relatif

**Level:** ⭐ BEGINNER

**Objective:** Accéder aux items par chemin

**Problem:**

Appelez une fonction module::function ou math::add.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Les chemins organisent la hiérarchie.
- `module::item` accède par chemin absolu ou relatif.
