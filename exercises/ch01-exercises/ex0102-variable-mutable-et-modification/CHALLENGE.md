# Variable mutable et modification

**Level:** ⭐ BEGINNER

**Objective:** Utiliser le mot-clé `mut` pour rendre une variable modifiable

**Problem:**

Déclarez une variable mutable, assignez-lui une valeur, puis modifiez-la deux fois.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- La mutabilité en Rust est explicite et locale à la variable.
- `let mut` autorise les modifications ultérieures.
