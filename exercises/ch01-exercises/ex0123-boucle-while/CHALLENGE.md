# Boucle while

**Level:** ⭐ BEGINNER

**Objective:** Répéter du code tant qu'une condition est vraie

**Problem:**

Comptez de 1 à 10 avec une boucle while.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- while vérifie la condition avant chaque itération.
- while condition { } continue tant que condition est true.
