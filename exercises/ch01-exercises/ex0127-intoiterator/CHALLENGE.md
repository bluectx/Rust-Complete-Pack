# IntoIterator

**Level:** ⭐ BEGINNER

**Objective:** Implémenter IntoIterator pour consommer une collection

**Problem:**

Convertissez une struct en itérateur avec IntoIterator.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Les trois variantes : iter(), iter_mut(), into_iter().
- IntoIterator = méthode .into_iter() pour consommer.
