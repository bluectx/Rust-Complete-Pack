# Custom iterador adapters

**Level:** ⭐ BEGINNER

**Objective:** Créer des adaptateurs d'itérateur personnalisés

**Problem:**

Écrivez une méthode .take_while(predicate) custom.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Composition d'itérateurs via le pattern wrapper.
- Les adaptateurs retournent des itérateurs wrappés.
