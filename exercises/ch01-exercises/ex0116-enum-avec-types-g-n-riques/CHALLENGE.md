# Enum avec types génériques

**Level:** ⭐ BEGINNER

**Objective:** Créer des enums paramétrés

**Problem:**

Définissez `enum Result<T, E> { Ok(T), Err(E) }` vous-même.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Comprenez la syntaxe derrière les enums du stdlib.
- Les enums peuvent être génériques.
