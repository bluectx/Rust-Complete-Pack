# Patterns de macros

**Level:** ⭐ BEGINNER

**Objective:** Utiliser différents patterns (expr, item, block, etc.)

**Problem:**

Créez une macro avec patterns multiples.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Chaque pattern capture différentes syntaxes.
- Patterns : expr, ty, item, block, stmt, pat, meta, ident.
