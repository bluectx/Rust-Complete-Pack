# Result::and_then et Result::or_else

**Level:** ⭐ BEGINNER

**Objective:** Utiliser des combinateurs Result avancés

**Problem:**

Chainez `.and_then()` et `.or_else()` pour du error handling monadique.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Pattern fonctionnel pour manipuler les Results.
- and_then combine Results ; or_else fournit une alternative.
