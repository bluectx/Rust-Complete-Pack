# Impl block génériques

**Level:** ⭐ BEGINNER

**Objective:** Écrire des méthodes génériques pour des structs génériques

**Problem:**

Écrivez `impl<T> MyStruct<T> { fn new(t: T) -> Self { } }`.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Combinaisons puissantes.
- Chaque type générique peut avoir des méthodes génériques.
