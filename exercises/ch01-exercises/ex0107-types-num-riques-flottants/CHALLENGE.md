# Types numériques flottants

**Level:** ⭐ BEGINNER

**Objective:** Utiliser f32 et f64 pour les nombres décimaux

**Problem:**

Créez deux variables flottantes, une f32 et une f64. Effectuez une division.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Le type par défaut pour un littéral flottant est f64.
- f32 (simple précision) et f64 (double précision).
