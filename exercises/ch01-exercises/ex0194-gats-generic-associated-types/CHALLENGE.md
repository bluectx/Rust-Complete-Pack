# GATs (Generic Associated Types)

**Level:** ⭐ BEGINNER

**Objective:** Utiliser les traits avec lifetimes sur les associated types

**Problem:**

Écrivez un trait avec `type Iter<'a>` et implémentez-le.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Maximum de flexibilité dans les traits.
- GATs = associated types génériques.
