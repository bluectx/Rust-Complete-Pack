# Type booléen

**Level:** ⭐ BEGINNER

**Objective:** Utiliser le type `bool` et les opérateurs logiques

**Problem:**

Créez des variables booléennes et combinez-les avec &&, ||, !

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Les opérateurs && (ET), || (OU), ! (NON).
- bool peut être true ou false.
