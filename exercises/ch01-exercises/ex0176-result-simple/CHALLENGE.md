# Result simple

**Level:** ⭐ BEGINNER

**Objective:** Comprendre Result<T, E> pour les opérations faillibles

**Problem:**

Écrivez une fonction qui retourne Result<i32, String> et appelez-la.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Result remplace les exceptions ; c'est explicite.
- `Result<T, E>` = Ok(T) ou Err(E).
