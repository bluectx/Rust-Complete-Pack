# Transfert d'ownership implicite

**Level:** ⭐ BEGINNER

**Objective:** Comprendre les scénarios de transfert non-évidents

**Problem:**

Créez une fonction qui prend ownership, puis transférez plusieurs fois.

**Example Runs:**

```bash
$ cargo run
# Output attendu selon votre implémentation
```

**Tests:**

Run: `cargo test`

All tests must pass.

**Key Learning Points:**

- Tracer l'ownership est essentiel au debugging.
- Ownership peut traverser plusieurs fonctions.
